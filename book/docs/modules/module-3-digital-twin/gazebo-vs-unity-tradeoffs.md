# Gazebo vs Unity Tradeoffs

## Learning Objectives

- Compare Gazebo and Unity for robotics simulation
- Choose appropriate platform for different use cases
- Understand strengths and limitations of each

## Comparison Matrix

| Aspect | Gazebo | Unity |
|---------|----------|--------|
| **Primary Use** | Physics simulation | Visual rendering |
| **ROS Integration** | Native | Via bridge |
| **Rendering** | Basic | Photorealistic |
| **Physics Accuracy** | High | Medium |
| **Cost** | Free | Free for non-commercial |
| **Asset Library** | Limited | Extensive |

## When to Use Gazebo

- Accurate physics required
- ROS 2 native integration needed
- Limited GPU resources

## When to Use Unity

- Photorealistic rendering critical
- Using existing Unity assets
- Training vision models
